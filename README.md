Haven — это анонимный веб-помощник для людей, столкнувшихся с домашним насилием или буллингом.

Основные возможности:

* Чат-помощник — отвечает на вопросы о безопасности, экстренных номерах, сборе "тревожной сумки"

* Режим прикрытия — маскируется под приложение для заметок

* Быстрый выход — двойное нажатие Escape или кнопка "Выход" => переход на нейтральный сайт

* План безопасности — возможность сохранить личный план действий

* Дыхательное упражнение 4-4-6 для успокоения

* Работает офлайн через Service Worker
